# MoviepilotSaubereSuche

Beim Eingeben eines Filmnamens in der Suchleiste wird ganz oben in den Suchergebnissen immer ein Film angezeigt, der nichts mit dem Suchtext zu tun hat. Dieses Browserscript entfernt den obersten Eintrag, damit gesuchte Filme schneller gefunden werden können. 

Download: [Github](https://github.com/Alsweider/MoviepilotSaubereSuche/releases/latest) | [Gist](https://gist.github.com/Alsweider/cd0400fbb0b2571ed82b2e6fd838c36f) | [Greasyfork](https://greasyfork.org/de/scripts/549199-moviepilot-saubere-suche)

# Installation

Um dieses Script zu nutzen, ist eine Browser-Erweiterung wie [Tampermonkey](https://www.tampermonkey.net/) nötig.
